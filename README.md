# Kolekcje

### W jaki sposób wygląda hierarchia interfejsów związana z kolekcjami?

Na szczycie hierarchi jest interface **Collection**.
Pochodnymi Collections są: 
* List, 
* Set, 
    *  Pochodna SortedSet
* Queue, 
* Deque.
Obok tej hierarchi znajduje się Map oraz pochodne SortedMap
